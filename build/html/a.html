集合と位相¶

集合¶

集合とは「ある特定の性質を備えたものの集まり」のことである。集合に含まれる1つ1つの「もの」を、その集合の要素または元という。 \(a\) が集合 \(A\) の元であることを

\[a\in A\]

で表し、 \(a\) は \(A\) に属するという。逆に \(a\) は \(A\) に属さないことは

\[a\notin A\]

のように表す。

集合は \(\{2,5,7\}\) のように、属する元を全て書き並べたり、 \(\{x|x\text{は条件}P\text{を満たす}\}\) のようにある条件を満たす元全体として定めることが多い。

集合 \(A\) の元のうち、条件 \(P\) を満たす元全体の集合を

\[\{x|x\in A,x\text{は条件}P\text{を満たす}\}\]

または

\[\{x\in A|x\text{は条件}P\text{を満たす}\}\]

で表す。

二つの集合 \(A,B\) について、 \(\forall a\in A:a\in B\) のとき、 \(A\) は \(B\) の部分集合であるといい、 \(A\subset B\) で表す。 この場合、 \(A\) は \(B\) に包まれる、または \(B\) は \(A\) を包むという。 \(A\subset B\) でないことを \(A\not\subset B\) で表す。

\(A\subset B \land A\ne B\) のとき、 \(A\) は \(B\) の真部分集合であるといい、 \(A\subsetneq B\) で表す。

議論している対象の要素全体の集合を全体集合または不偏集合という。 全体集合を \(U`で表す時、議論の対象となる集合 :math:`A\) を任意に選ぶと

\[A\subset U \Leftrightarrow \forall x\in A : x\in U\]

が成り立つ。

要素を一つも含まない集合を空集合といい、 \(\emptyset\) で表す。

全体集合が \(U\) であるとき、空集合は内包的に

\[\emptyset = \{x\in U \mid x \neq x\}\]

と表すことができる。

元を有限個しか含まない集合および空集合を有限集合といい、そうでない集合を無限集合と呼ぶ。

集合演算¶

集合論では集合を被演算子と呼ばれる演算を定義する。集合演算で使われる被演算子は \(c,\cup,\cap,\setminus,\Delta\) などであり、これらを集合演算子と呼ぶ。 ここから特に断らない限り \(U\) を全体集合、 二つの集合 \(A,B\) について \(A\subset U,B\subset U\) とする。

\[A^c = \{x\in U \mid x\notin A\}\]

を \(A\) の補集合という。

\[A\cup B = \{x\in U \mid x\in A \lor x\in B\}\]

を \(A\) と \(B\) の和集合といい、

\[A\cap B = \{x\in U \mid x\in A \land x\in B\}\]

を \(A\) と \(B\) の共通部分といい、

\[A\setminus B = \{x\in U \mid x\in A \land x\notin B\}\]

を \(A\) と \(B\) の差集合という。

\[A\Delta B = (A\setminus B)\cup (B\setminus A)\]

を \(A\) と \(B\) の対称差という。

写像¶

集合 \(A,B\) に対し、 \(A\) のすべての元に対して \(B\) の元が唯一定まる時、その対応を写像と呼ぶ。

\(f:A\to B\) で、 \(f\) が \(A\) から \(B\) への写像であることを表す。 \(f:a\mapsto b\) で、 写像 \(f\) によって \(a\in A\) が \(b\in B\) に移ることを表す。

\[\forall a\in A, \exists !b \in B : b=f(a)\]

\(A\) を \(f\) の始域、 \(B\) を \(f\) の終域という。また \(b\) を \(f\) による \(a\) の像といい、 \(a\) を \(f\) による \(b\) の原像という。

全射は \(\forall b\in B,\exists a\in A:b=a\)

単射は \(\forall a_1,a_2\in A:(b_1=b_2\Rightarrow a_1=a_2)\)
