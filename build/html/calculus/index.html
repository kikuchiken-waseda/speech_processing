実数の集合¶

区間¶

\(a,b\in\mathbb{R}(a<b)\) に対し、次の形の集合を区間という。

開区間

\[(a,b) :=\{x\in\mathbb{R}\mid a<x<b\}\]

閉区間

\[[a,b] :=\{x\in\mathbb{R}\mid a\leq x\leq b\}\]

ただし、 数列 \((-\infty,\infty)\in\mathbb{R}\) は開区間と閉区間でもあると考えることとする。

近傍¶

\(\varepsilon>0,a\in\mathbb{R}\) に対して

\[U_{\varepsilon}(a) := \{x\in\mathbb{R}\mid |x-a| <\varepsilon\}=(a-\varepsilon,a+\varepsilon)\]

とおく。この開区間 \(U_{\varepsilon}(a)\) を点 \(a\) の \(\varepsilon\) 近傍(または単に近傍)という。

上界・下界¶

\[ \begin{align}\begin{aligned}\forall x\in X,\exists a\in\mathbb{R}:\\a\geq x\end{aligned}\end{align} \]

のとき、 \(a\) が \(X\) の上界であるという。

\[ \begin{align}\begin{aligned}\forall x\in X,\exists a\in\mathbb{R}:\\a\leq x\end{aligned}\end{align} \]

のとき、 \(a\) が \(X\) の下界であるという。

最大値・最小値¶

\(a\in\mathbb{R}\) が \(X\) の上界かつ、 \(a\in X\) を満たすとき、 \(a\) を \(X\) の最大値といい、 \(\max X\) で表す。 \(a\in\mathbb{R}\) が \(X\) の下界かつ、 \(a\in X\) を満たすとき、 \(a\) を \(X\) の最小値といい、 \(\min X\) で表す。

上限・下限¶

\(X\) の上界が空集合でない時、上界の最小値を上限といい、 \(\sup X\) と表す。 上界が空集合の場合、上限がないといい、 \(\sup X = \infty\) と表すことがある。

\(X\) の下界が空集合でない時、下界の最大値を下限といい、 \(\inf X\) と表す。 下界が空集合の場合、下限がないといい、 \(\inf X = -\infty\) と表すことがある。

数列の極限¶

数列 \(\{a_n\}^{\infty}_{n=1}\) と \(\alpha\in\mathbb{R}\) について、

\[ \begin{align}\begin{aligned}\forall\varepsilon>0,\exists N(\varepsilon)\in\mathbb{N}:\\n\geq N(\varepsilon)\Rightarrow|a_n-\alpha|<\varepsilon\end{aligned}\end{align} \]

このとき、数列 \(\{a_n\}^{\infty}_{n=1}\) の極限値は \(\alpha\) であるといい、

\[\lim_{n\to\infty}a_n=\alpha\text{ または } a_n\to\alpha(n\to\infty)\]

と表す。このとき、数列 \(\{a_n\}^{\infty}_{n=1}\) は \(\alpha\) に収束するという。

数列 \(\{a_n\}^{\infty}_{n=1}\) がどのような \(\alpha\in\mathbb{R}\) にも収束しないとき、発散するという。 さらに、

\[ \begin{align}\begin{aligned}\forall K>0,\exists N(K)\in\mathbb{N}:\\n\geq N(K)\Rightarrow a_n>K\end{aligned}\end{align} \]

のとき、数列 \(\{a_n\}^{\infty}_{n=1}\) は \(\infty\) に発散するといい、

\[\lim_{n\to\infty}a_n=\infty\]

と表し、

\[ \begin{align}\begin{aligned}\forall K>0,\exists N(K)\in\mathbb{N}:\\n\geq N(K)\Rightarrow a_n<K\end{aligned}\end{align} \]

のとき、数列 \(\{a_n\}^{\infty}_{n=1}\) は \(-\infty\) に発散するといい、

\[\lim_{n\to\infty}a_n=-\infty\]

と表す。

級数¶

数列 \(\{a_n\}^{\infty}_{n=1}\) の各項を足し合わせた

\[a_1+a_2+a_3+\cdots+a_n+\cdots\]

を \(\{a_n\}^{\infty}_{n=1}\) の作る級数と言い、 \(\sum_{n=1}^{\infty}{a_n}\) と表す。

\(S_n=\sum^{n}_{k=1}a_k\) を第 \(n\) 部分和といい、 数列 \(\{S_n\}^{\infty}_{n=1}\) が収束するか発散するかに応じて、 級数 \(\sum_{n=1}^{\infty}{a_n}\) は収束するまたは発散するという。 部分和が収束する場合に \(S=\lim_{n\to\infty}S_n\) を級数の和といい、 \(\sum_{n=1}^{\infty}{a_n}=S\) と書く。 また、 \(S=\pm\infty\) となる場合も、それぞれ \(\sum_{n=1}^{\infty}{a_n}=\pm\infty\) で表す。

単調数列¶

数列 \(\{a_n\}^{\infty}_{n=1}\) が単調増加であるとは、

\[\forall n\in \mathbb{N} : a_n\leq a_{n+1}\]

が成り立つことである。特に

\[\forall n\in \mathbb{N} : a_n < a_{n+1}\]

が成り立つとき、狭義単調増加である。

数列 \(\{a_n\}^{\infty}_{n=1}\) が単調減少であるとは、

\[\forall n\in \mathbb{N} : a_n\geq a_{n+1}\]

が成り立つことである。特に

\[\forall n\in \mathbb{N} : a_n > a_{n+1}\]

が成り立つとき、狭義単調減少である。

単調増加または単調減少な数列をまとめて単調数列という。

Napier数¶

\[e := \lim_{n\to\infty}(1+\frac{1}{n})^n\]

とし、これをNapier数という。

部分列¶

\(n_k \in \mathbb{N}\) を項とする、狭義単調増加数列 \(\{n_k\}^{\infty}_{k=1}\) をとる。 数列 \(\{a_n\}^{\infty}_{n=1}\) に対して、数列 \(\{a_{n_k}\}^{\infty}_{k=1}\) をその部分列という。

関数の極限¶

少なくとも、点 \(a\) の近傍から点 \(a\) を除いたところで定義されている関数 \(f(x)\) と定数 \(\alpha\in\mathbb{R}\) について、

\[ \begin{align}\begin{aligned}\forall\varepsilon>0,\exists \delta(\varepsilon)>0:\\0 \neq |x-a| <\delta(\varepsilon) \Rightarrow |f(x)-\alpha| <\varepsilon\end{aligned}\end{align} \]

を満たすとき、関数 \(f(x)\) の点 \(a\) での極限値は \(\alpha\) であるといい

\[\lim_{x\to a}f(x)=\alpha\text{ または } f(x)\to\alpha(x\to a)\]

で表す。このとき、関数 \(f(x)\) は点 \(a\) で \(\alpha\) に収束するという。

一方、関数 \(f(x)\) が点 \(a\) でどのような定数 \(\alpha\in\mathbb{R}\) にも収束しない時、発散するという。

\[ \begin{align}\begin{aligned}\forall K>0,\exists \delta(K)>0:\\0 \neq |x-a| <\delta(K) \Rightarrow f(x)>K\end{aligned}\end{align} \]

の時、 f(x) は点 \(a`で :math:\)infty` で発散するといい

\[\lim_{x\to a}f(x)=\infty\]

で表す。また

\[ \begin{align}\begin{aligned}\forall K>0,\exists \delta(K)>0:\\0 \neq |x-a| <\delta(K) \Rightarrow f(x)<K\end{aligned}\end{align} \]

の時、 \(f(x)\) は点 \(a\) で \(-\infty\) で発散するといい

\[\lim_{x\to a}f(x)=-\infty\]

で表す。

少なくとも、ある \(c\) に対し区間 \((c,\infty)\) 定義されている関数 \(f(x)\) と定数 \(\alpha\in\mathbb{R}\) について、

\[ \begin{align}\begin{aligned}\forall\varepsilon>0,\exists L(\varepsilon)>0:\\x>L(\varepsilon) \Rightarrow |f(x)-\alpha| <\varepsilon\end{aligned}\end{align} \]

を満たすとき、関数 \(f(x)\) の \(x\to\infty\) での極限値は \(\alpha\) であるといい

\[\lim_{x\to \infty}f(x)=\alpha\text{ または } f(x)\to\alpha(x\to \infty)\]

で表す。このとき、関数 \(f(x)\) は \(x\to\infty\) で \(\alpha\) に収束するという。

一方、関数 \(f(x)\) が \(x\to\infty\) でどのような定数 \(\alpha\in\mathbb{R}\) にも収束しない時、発散するという。

\[ \begin{align}\begin{aligned}\forall K>0,\exists L(K)>0:\\x<L(K) \Rightarrow f(x)>K\end{aligned}\end{align} \]

の時、 f(x) は \(x\to\infty\) で \(\infty\) で発散するといい

\[\lim_{x\to \infty}f(x)=\infty\]

で表す。また

\[ \begin{align}\begin{aligned}\forall K>0,\exists L(K)>0:\\x<L(K) \Rightarrow f(x)<K\end{aligned}\end{align} \]

の時、 \(f(x)\) は \(x\to\infty\) で \(-\infty\) で発散するといい

\[\lim_{x\to \infty}f(x)=-\infty\]

で表す。

\(x\to -\infty\) での極限についても同様に定義する。

微分係数¶

関数 \(f(x)\) が点 \(a\) の近傍で定義されていて、極限値

\[\lim_{x\to a} \frac{f(x)-f(a)}{x-a}\]

が存在する時、この極限値を \(f(x)\) の点 \(a\) における微分係数といい

\[f'(a) := \lim_{x\to a} \frac{f(x)-f(a)}{x-a}\]

で表す。この時、 \(f(x)\) は点 \(a\) で微分可能であるという。

\(h = x-a\) とおくと

\[f'(a) = \lim_{h\to 0} \frac{f(a+h)-f(a)}{h}\]

で表すことができる。

接線¶

関数 math`f(x)` は点 \(a\) で微分可能とする。この時、直線

\[y = f'(a)(x-a) + f(a)\]

を曲線 \(y=f(x)\) 上の点 \((a,f(a))\) における接線という。

導関数¶

開区間 \(I\) で定義された関数 \(f(x)\) が \(I\) の全ての点で微分可能であるとき、 \(f(x)\) は \(I\) で微分可能であるという。

\[f'(x) := \lim_{h\to 0} \frac{f(x+h)-f(x)}{h} (x\in I)\]

を \(f(x)\) の導関数という。 1次導関数とも呼ばれる。 \(y=f(x)\) の導関数は

\[f^{(1)}(x),\frac{d}{dx}f(x),y',y^{(1)},\frac{dy}{dx}\]

のようにも表される。

\(I\) が端点を持つ場合、面倒なので後でかく。

高次導関数¶

区間 \(I\) で定義された関数 \(f(x)\) が \(I\) で微分可能で、\((n-1)\) 次導関数 \(f^{(n-1)}(x)\) も \(I\) で微分可能であるとき、 \(f\) はn回微分可能であるという。 この時、 \((n-1)\) の導関数を \(f\) の \(n\) 次導関数といい、 \(y=f(x)\) の \(n\) 次導関数を

\[f^{(n)}(x),\frac{d^n}{dx^n}f(x),y^{(n)},\frac{d^ny}{dx^n}\]

などで表す。

積分法¶

区間の分割¶

開区間 \(I=[a,b]\) に対して、次のような \(I\) の分点の組

\[\Delta : a = x_0<x_1<x_2<\cdots<x_{n-1}<x_n = b\]

を \(\Delta = \{x_k\}^{n}_{k=0}\) で表し、 \(I\) の分割という。ここで、\(n\in\mathbb{N}\) も自由に選んで良い。 また \(I\) の分割 \(\Delta = \{x_k\}^{n}_{k=0}\) に対して

\[|\Delta| = \max\{x_k - x_{k-1} \mid k = 1,2,\cdots,n\}\]

を分割 \(\Delta\) の幅という。

リーマン和¶

\(f(x)\) を開区間 \(I=[a,b]\) で有界な関数とする。 \(I\) の分割 \(\Delta = \{x_k\}^{n}_{k=0}\) 代表点 \(\xi_k\in[x_{k-1},x_k]\) を決めた時

\[S(f;\Delta,\{\xi_k\}) = \sum_{k=1}^{n}f(\xi_k)(x_k-x_{k-1})\]

を \((\Delta,\{\xi_k\})\) に関する \(f(x)\) のリーマン和という。

リーマン積分¶

関数 \(f(x)\) は有界閉区間 \(I=[a,b]\) で有界であるとする。この時ある実数 \(\alpha\) が存在して、任意の代表点 \(\{\xi_k\}\)

\[\lim_{|\Delta|\to 0} S(f;\Delta,\{\xi_k\}) = \alpha\]

が成り立つ時、 \(f(x)\) は \(I\) で積分可能であるという。

このとき、

\[\int^{b}_{a}f(x)dx = \alpha\]

を \(f(x)\) の \(I\) でのリーマン積分という。

原始関数¶

区間 \(I\) 上の関数 \(f(x)\) に対して

\[\frac{d}{dx}F(x) = f(x) (x\in I)\]

を満たす関数 \(F(x)\) が存在する時、 \(F(x)\) を \(f(x)\) の原始関数という。

不定積分¶

関数 \(f(x)\) が区間 \(I\) に含まれる任意の有界閉区間上で積分可能とする。この時、 \(a\in I\) と定数 \(C\) に対して、関数

\[F(x) = \int^{x}_{a}f(t)dt+C (x\in I)\]

を定める。この \(F(x)\) を記号

\[\int f(x)dx\]

で表し、区間 \(I\) における \(f(x)\) 不定積分という。

微積分学の基本定理¶

関数 \(f(x)\) が区間 \(I\) 上で連続ならば、 \(a\in I\) とすれば

\[\frac{d}{dx}\int^{x}_{a}f(t)dt = f(x) (x\in I)\]

が成り立つ。

証明は面倒なので省略。
