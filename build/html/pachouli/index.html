マクローリン展開¶

マクローリン展開はテイラー展開の \(a=0\)

\[f(x) = \sum^{\infty}_{k=0}f^{(k)}(0)\frac{x^k}{k!}\]

これはテイラー展開

\[f(x) = \sum^{\infty}_{k=0}f^{(k)}(a)\frac{(x-a)^k}{k!}\]

テイラーの定理

\(f\) が \([a,x]\) 上 \(n-1\) 回連続微分可能かつ、 \((a,x)\) 上で \(n\) 回微分可能であるとき、

\[f(x) = \sum^{n-1}_{k=0}f^{(k)}(a)\frac{(x-a)^k}{k!} + f^{(n)}(c)\frac{(x-a)^n}{n!}\]

なる \(c(a<c<x)\) が存在する。 ここで、最終項 \(R_n = \frac{f^{(n)}(c)}{n!}(x-a)^n\) をラグランジュの剰余項という。
