論理¶

かつ: \(\land\)

または: \(\lor\)

\(A := B\) としたら \(A\) を \(B\) で定義する。 また、集合 \(\{x\in S|x\text{は条件}P\text{を満たす}\}\) に名前をつけることを、 \(A:=\{x\in S|x\text{は条件}P\text{を満たす}\}\) で表す。

量化¶

全称量化子: \(\forall\)

存在量化子: \(\exists\)

全ての \(x\) が条件 \(P\) を満たす(\(\text{For all }x,P\) )ことを

\[\forall x : P\]

で表す。

\(P\) を満たす \(x\) が存在する(\(\text{There exists }x\text{ such that }P\) )ことを

\[\exists x : P\]

で表す。

他にも色々示し方があるので注意するように。

集合¶

集合とは「ある特定の性質を備えたものの集まり」のことである。集合に含まれる1つ1つの「もの」を、その集合の要素または元という。 \(a\) が集合 \(A\) の元であることを

\[a\in A\]

で表し、 \(a\) は \(A\) に属するという。逆に \(a\) は \(A\) に属さないことは

\[a\notin A\]

のように表す。

集合は \(\{2,5,7\}\) のように、属する元を全て書き並べたり、 \(\{x|x\text{は条件}P\text{を満たす}\}\) のようにある条件を満たす元全体として定めることが多い。

集合 \(A\) の元のうち、条件 \(P\) を満たす元全体の集合を

\[\{x|x\in A,x\text{は条件}P\text{を満たす}\}\]

または

\[\{x\in A|x\text{は条件}P\text{を満たす}\}\]

で表す。

要素を一つも含まない集合を空集合といい、 \(\emptyset\) で表す。

元を有限個しか含まない集合および空集合を有限集合といい、そうでない集合を無限集合と呼ぶ。

二つの集合 \(A,B\) について、 \(\forall a\in A:a\in B\) のとき、 \(A\) は \(B\) の部分集合であるといい、 \(A\subset B\) で表す。 この場合、 \(A\) は \(B\) に包まれる、または \(B\) は \(A\) を包むという。 \(A\subset B\) でないことを \(A\not\subset B\) で表す。

\(A\subset B \land A\ne B\) のとき、 \(A\) は \(B\) の真部分集合であるといい、 \(A\subsetneq B\) で表す。

二つの集合 \(A,B\) について、

\[A\cup B = \{x|x\in A \lor x\in B\}\]

を \(A\) と \(B\) の和集合といい、

\[A\cap B = \{x|x\in A \land x\in B\}\]

を \(A\) と \(B\) の共通部分といい、

\[A\setminus B = \{x|x\in A \land x\notin B\}\]

を \(A\) と \(B\) の差集合という。

議論している対象の要素全体の集合を全体集合または不偏集合といい、 \(U\) を全体集合、 \(A\subset U\) とするとき

\[A^c = \{x\in U |x\notin A\}\]

写像¶

集合 \(A,B\) に対し、 \(A\) のすべての元に対して \(B\) の元が唯一定まる時、その対応を写像と呼ぶ。

\(f:A\to B\) で、 \(f\) が \(A\) から \(B\) への写像であることを表す。 \(f:a\mapsto b\) で、 写像 \(f\) によって \(a\in A\) が \(b\in B\) に移ることを表す。

\(A\) を \(f\) の始域、 \(B\) を \(f\) の終域という。また \(b\) を \(f\) による \(a\) の像といい、 \(a\) を \(f\) による \(b\) の原像という。

全射は \(\forall b\in B,\exists a\in A:b=a\)

単射は \(\forall a_1,a_2\in A:(b_1=b_2\Rightarrow a_1=a_2)\)

後で付け足す。

関数¶

\(X=\{x\in\mathbb{R}|X\ne\emptyset\}\) とする。

写像 \(f:X\to \mathbb{R}\) を関数と言う。

このとき、 \(x\in X\) について \(f:x\mapsto y\) を満たす \(y\in\mathbb{R}\) を \(y=f(x)\) と表す。 \(X\) を \(f\) の定義域、 \(f(X)=\{f(x)|x\in X\}\) を \(f\) の値域という。

関数を \(y=f(x)\) と表したとき、 \(x\) を \(f\) の独立変数、 \(y\) を \(f\) の従属変数という。

値域 \(f(X)\) の最大値、最小値が存在するとき、それらを関数 \(f\) の最大値、最小値という。

絶対値¶

\(x\in\mathbb{R}\) の絶対値 \(|x|\) を次のように定める。

\[ \begin{align}\begin{aligned}|x| := \begin{cases} x \:\;\;\;\;\text{if }x\geq0\\ -x \;\;\text{if }x<0 \end{cases}\end{aligned}\end{align} \]

ガウス記号¶

\(x\in\mathbb{R}\) に対し、

\[[x] := \text{max}\{n\in\mathbb{Z}|n\leq x\}\]

の \([\,\cdot\,]\) をガウス記号という。
